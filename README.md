# 🦸 英雄作业计时器

> 小朋友作业挑战计时器 — 纯前端单文件，无需任何依赖

**在线访问**：https://hero-timer-king.vercel.app

## 功能

### 核心玩法
- ⏱️ 作业计时挑战 — 按下开始，写完按挑战成功
- ⚡ 能量水槽系统 — 每完成5次挑战解锁超级大奖
- 🏆 排行榜 — 与虚拟小伙伴比拼
- 🎖️ 终极大奖 — 兑换码奖励机制（30种真实奖励）

### 账号系统
- 📝 注册/登录 — 多账号独立数据
- 🔒 数据隔离 — 每个账号的打卡、等级、成就完全独立
- 🚪 登出切换 — 支持多用户共用同一设备

### 游戏化系统
- ⭐ XP/等级系统 — 25级成长路线，从"实习小英雄"到"传说英雄"
- 🏅 成就墙 — 14个成就（里程碑/速度/特殊类）
- 🎖️ 勋章展示柜 — 4枚勋章×4阶进化（铜→银→金→钻石）
- 🔥 连续打卡 — 连续天数追踪，心形粒子特效
- 📋 每日任务 — 3个每日任务，完成奖励XP

### 数据可视化
- 📅 周日历打卡 — 本周7天打卡状态
- 📊 柱状图 — 近7天专注时间趋势
- 🏅 个人记录墙 — 最佳时间/总次数/总专注/连续打卡
- 📰 英雄周报 — 每周一自动生成周报，对比上周

### 家长功能
- 👨‍👩‍👧 家长看板 — 密码保护的数据面板
- 📈 专注趋势图 — 近7天柱状图
- 🎫 奖品兑换历史 — 兑换码状态追踪

### 个性化
- 🎨 3套主题 — 海洋/太空/森林（按等级解锁）
- 😊 20个头像 — 按等级逐步解锁
- 🦸 昵称设置 — 自定义英雄昵称

### 智能提醒
- 😴 休息提醒 — 专注45分钟自动提醒
- ⏳ 间隔提醒 — 两次挑战间隔5分钟提醒
- 💬 动态鼓励 — 里程碑弹窗 + 加油文字轮播
- 🎉 升级特效 — 升级时全屏动画 + 彩纸

## 技术特点

- 纯前端单文件 `index.html`，零依赖零构建
- 北京时间 (UTC+8) 精确显示，全球可用
- localStorage 持久化，账号数据隔离
- 响应式适配：桌面/平板/手机（底部工具栏）
- 毛玻璃 + 3D拟物按钮 + 氛围粒子动效
- Google Fonts Nunito 儿童友好字体

## 开发

直接打开 `index.html` 即可使用。
